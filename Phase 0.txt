Phase 0 — The Build Plan
Step 1: Environment Setup (Day 1)

    Install Godot 4.x
    Install Ink (Inky editor) + godot-ink plugin
    Create project structure
    Get a blank scene running

Step 2: The Psyche State System (Day 2-3)

    Code the Voice state model in GDScript (Level, Attention, Integrity, Status for 3 voices)
    Implement the cultivation mechanic (choice → voice gains/losses)
    Implement threshold triggers (distortion, silence)

Step 3: The Dialogue System (Day 4-5)

    Write the MVP Ink script — three voices reacting to one dilemma
    Connect Ink variables to the psyche state system
    Test: does a choice change state? Do voices respond differently based on state?

Step 4: The World Response (Day 6-8)

    Build one Garden of Memory room (placeholder art — even colored rectangles)
    Implement one landscape-to-psyche mapping rule (e.g., Curiosity ↑ = fog lifts)
    Make the world visibly respond to a player choice

Step 5: The Question Object + Future Self Echo (Day 9-10)

    Design the one philosophical dilemma (the question object)
    Write the future self's single line of dialogue
    Implement the Night phase transition

Step 6: Playtest Loop (Day 11-14)

    Connect Morning → Afternoon → Evening → Night flow
    Play it. Feel it. Does it create feeling?
    Iterate based on emotional response, not feature checklist
